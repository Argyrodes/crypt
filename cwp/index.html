Crypt Writer

Minimalist, yet almost feature complete writer and editor.



CRYPT WRITING STANDARDS

Headings are distinguished by UPPER CASE characters.
Emphasis is distinguished by a * at the beginning and the end of the intended word
** at the end of a sentence signifies further writings/notes found in the annotations section.

*Note: Autosave may not work if your browser does not allow access to localstorage.*
to ensure autosave works you can use the desktop build by:

1. pulling this repo
2. open a terminal and cd the directory
3. type 'npm install'
4. type 'npm start'
